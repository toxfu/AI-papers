# AI, the Way

Repositorio de estudio y documentación sobre arquitecturas Transformer y temas relacionados en Deep Learning, con notebooks explicativos, visualizaciones y publicación web con MyST.

## Objetivo

Este proyecto está orientado a:

- estudiar papers clave de IA con enfoque práctico,
- documentar conceptos técnicos en formato notebook,
- combinar teoría, implementación y visualización,
